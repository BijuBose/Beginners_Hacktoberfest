What is Hacktoberfest?
Hacktoberfest is a month long virtual festival event to celebrate open source contributions.
It is the easiest way to get into open source!
During the entire month of October 2019, all you have to do is contribute to any open source projects and open at least 4 pull requests. Yes, any project and any kind of contributions. 3. Type in some text and commit the changes
